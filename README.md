## 토마토 Pro/Master 페이지 [수정하는 방법]

1. 아래 [제미나이 수정링크]를 통해 페이지 내용 수정

2. 제미나이 통해 수정하고, 제미나이 내 화면 우측상단 [공유] > [콘텐츠복사]

3. 깃허브 > pro-master 레포지토리 > index.html에 들어가서 붙여넣기 후 commit changes 클릭
